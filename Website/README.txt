# Healthcare Database System

This project is a PHP, HTML, and SQL-based website developed using XAMPP. It provides a platform for managing healthcare-related data.

## Getting Started

To run this website on your local machine, follow these steps:

### Prerequisites

- XAMPP installed on your system.

### Installation

1. Clone this repository to your local machine.

   git clone https://github.com/Fliperdudole/Healthcare-Database-System.git

2. Navigate to the htdocs directory in your XAMPP installation folder.

   cd /path/to/xampp/htdocs

3. Copy the contents of the cloned repository into the htdocs directory.

4. Start the Apache and MySQL servers using the XAMPP control panel.

5. Import the SQL database:
   - Open phpMyAdmin by navigating to http://localhost/phpmyadmin in your web browser.
   - Create a new database named healthcare_database.
   - Import the SQL file database.sql located in the Database directory of this repository into the healthcare_database.

### Usage

1. Open your web browser and navigate to http://localhost/Healthcare-Database-System/Website/login/login.html.
2. You will be directed to the login page of the Healthcare Database System.
